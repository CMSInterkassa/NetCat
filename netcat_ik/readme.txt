Модуль предназначен для CMS NetCat 5.8.x
Версия: 1.4


Для установки в ручном режиме платежного модуля Interkassa необходимо произвести следующие действия:

1. Убедитесь в соответствии версий модуля и вашей CMS, они должны совпадать.

2. Скопировать interkassa.php в /netcat/modules/payment/classes/system/
3. Скопировать папку interkassa в /netcat_template/template/

4. Выполнить SQL запросы из файла install.sql (Инструменты -> Командная строка SQL) /netcat/modules/payment/

5.В админ панели перейдите в следующий раздел Настройки -> Прием платежей -> Интеркасса и настройте ее:
*Все настройки можно взять на сайте Интеркассы(идентификатор магазина в списке ваших кас, тестовый и секретный ключ во вкладке безопасность)

	test_mode	- Тестовый режим
	id_cashbox	- Идентификатор вашей кассы на сайте Интеркассы
	secret_key	- Секретный ключ
	test_key	- Тестовый ключ
	api_enable	- Вкл/Выкл API
	api_id		- Идентификатор API
	api_key		- Ключ API
	PaymentSuccessPage	- URL страницы при Успешной оплате
	PaymentFailedPage	- URL страницы при Неуспешной оплате
	PaymentPendingPage	- URL страницы при Ожидании оплате

	Что бы включить тествый режим заполните поле test_mode единицей, иначе оставте поле пустым
	Что бы Вкл. API заполните поле api_enable единицей, иначе оставте поле пустым

6. В меню «Оплата» добавьте новый вариант оплаты. Или добавьте к уже существующему способу.

7. В настройках вашей кассы на сайте Интеркассы ,в разделе Интерфейс:

   Поставьте галочки "Разрешить переопределять в запросе" для полей - "URL Успешной оплаты", "URL Неуспешной оплаты", "URL Ожидания оплаты", "URL Взаимодействия"

8. В целях безопасности советуем во вкладке Безопасность включить проверку цифровой подписи.

!!!ОБЯЗАТЕЛЬНО ОЗНАКОМИТЬСЯ
Для тестирования платежей в кассе используйте тестовую валюту XTS (тестовая платежная система), для это необходимо включить в кассе этот пункт, 
и далее при перенаправлении на сайт "Интеркассы" выбрать ее в качестве оплаты, 
далее система сгенерирует тестовую страничку со всеми видами ответа от сервера, это поможет вам настроить все необходимые параметры и убедиться в работоспособности вашей системы.
